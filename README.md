### 📌 Featured Project

**[Groundwater Lithium & Dementia Prevalence in Texas](https://chloedbarker.github.io/capstone/)** — MS Capstone, *Best Presentation Award*

An end-to-end epidemiological data science study investigating whether naturally occurring groundwater lithium concentrations are associated with county-level dementia prevalence across Texas — using OLS regression with HC3 robust standard errors, permutation testing, bootstrap confidence intervals, and geospatial analysis.

`Python` `GeoPandas` `OLS · HC3 Robust SEs` `Permutation Testing` `Bootstrap CIs` `Public Health Research`
